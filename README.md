# Automated Invoice Tracking Dashboard

## Project Overview
This Excel dashboard automates invoice tracking and helps monitor overdue payments.

## Features
- Automatic Invoice ID
- Invoice Status (Paid, Pending, Overdue)
- Due-Date Alerts
- Days Overdue Calculation
- Interactive Dashboard
- Pie Chart and Bar Chart
- Conditional Formatting

## Excel Functions Used
- IF
- TODAY
- COUNTIF
- SUM
- TEXT
- RIGHT
- UPPER

## Tools Used
- Microsoft Excel
